# PRD-002: 메인 대시보드

## 배경

인증 후 사용자가 처음 보는 화면. 전체 자산 현황을 한눈에 파악하고, 매매 이벤트를 기록하거나 CSV로 불러올 수 있어야 한다. 한국 주식 / 미국 주식 / 코인을 통합해서 보여주되 부문별 구분도 유지한다.

## 목표

- 누적 자산 변화를 꺾은선 그래프로 시각화
- 부문별 자산 비중을 파이 차트로 표시
- 보유 종목 카드를 한눈에 확인 (실시간 현재가 기준 수익률 포함)
- 매매 이벤트를 수동 입력 또는 증권사 CSV 업로드로 등록

## 범위

### 포함
- 메인 대시보드 화면 (히스토리 그래프 → 파이 차트 → 종목 카드 리스트)
- 매매 이벤트 수동 입력
- 증권사 CSV 파일 업로드 및 파싱
- 현재가 실시간 API 연동 (한국 주식 / 미국 주식 / 코인)
- 환율 자동 조회 (실시간 또는 주기적 폴링)
- 부문별 필터 (전체 / 한국 주식 / 미국 주식 / 코인)

### 제외
- 종목별 상세 차트 (추후 추가)
- 매매 이벤트 달력 뷰 및 메모 기능 (추후 추가)
- 증권사 API 직접 연동 (추후 추가)

## 화면 구성 및 레이아웃

### 메인 대시보드 (`DashboardScreen`)

스크롤 구조. 상단부터 순서대로:

```
[부문 필터 탭] 전체 | 한국주식 | 미국주식 | 코인

[누적 자산 히스토리 그래프]
  - 꺾은선 그래프 (3개 라인)
  - 기간 전환: 일 / 주 / 월 / 년

[부문별 자산 비중 파이 차트]
  - 한국주식 / 미국주식 / 코인 비중 (%)
  - 범례 포함

[종목 카드 리스트]
  - 부문별 그룹 헤더
  - 각 종목 카드
```

우하단 FAB(+) 버튼 → 매매 이벤트 입력 바텀시트

---

### 누적 자산 히스토리 그래프

**3개 라인**

| 라인 | 내용 |
|------|------|
| 누적 총 자산 | 예수금 + 전 종목 평가금액 합산 |
| 누적 투입 원금 | 매수 정산금액 누계 - 매도 정산금액 누계 |
| 누적 수수료 + 세금 | 전 거래 수수료 + 세금 누계 |

**기간 전환**

| 단위 | X축 표시 |
|------|---------|
| 일 | 시간 (오전 9시 ~ 오후 6시) |
| 주 | 요일 (월 ~ 금) |
| 월 | 날짜 (1일 ~ 말일) |
| 년 | 월 (1월 ~ 12월) |

**부문 필터 연동**: 상단 탭 선택 시 해당 부문 자산만 반영된 그래프로 전환. "전체" 선택 시 통합.

---

### 부문별 자산 비중 파이 차트

- 현재 평가금액 기준 (원화 환산)
- 한국주식 / 미국주식 / 코인 3개 섹터
- 각 섹터에 비중(%) 및 금액 표시
- 부문 필터가 "전체"일 때만 표시. 특정 부문 선택 시 숨김.

---

### 종목 카드

카드 1장이 보유 종목 1개를 나타낸다.

**표시 정보**

| 항목 | 내용 |
|------|------|
| 종목명 + 티커 | 예: 삼성전자 005930 |
| 자산 유형 배지 | 한국주식 / 미국주식 / 코인 |
| 보유 수량 | |
| 평균 매수가 | 원화 표시 (미국주식은 달러 + 원화 병기) |
| 현재가 | 달러 + 원화 병기 (미국주식/코인) |
| 평가금액 | 현재가 × 보유수량 (원화 환산) |
| 수익률 | (현재가 - 평균매수가) / 평균매수가 × 100 (%) |
| 평가손익 | 평가금액 - 투입 원금 (원화, + / - 색상 구분) |

---

### 매매 이벤트 입력 (바텀시트)

**입력 필드**

| 필드 | 타입 | 비고 |
|------|------|------|
| 자산 유형 | 선택 | 한국주식 / 미국주식 / 코인 |
| 매매 유형 | 선택 | 매수 / 매도 |
| 종목명 | 텍스트 | |
| 티커 | 텍스트 | 예: 005930, AAPL, BTC |
| 체결 날짜 | 날짜 선택 | |
| 수량 | 숫자 | |
| 체결가 (1주당) | 숫자 | |
| 통화 | 선택 | KRW / USD (코인은 티커 기반 자동) |
| 수수료 | 숫자 | |
| 수수료 통화 | 선택 | 기본 KRW, 코인은 해당 티커 선택 가능 |
| 세금 | 숫자 | |
| 정산금액 | 자동 계산 | 체결가 × 수량 ± 수수료 ± 세금. 수동 수정 가능 |

**CSV 업로드**
- 바텀시트 상단에 "CSV 파일 업로드" 버튼
- 증권사별 파일 포맷을 서버에서 파싱
- 지원 증권사는 파일 수령 시 순차 추가
- 업로드 후 파싱 결과 미리보기 → 확인 후 저장

## 데이터 모델 (개요)

### 매매 이벤트 (`trade_events`)

| 컬럼 | 타입 | 내용 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | FK → auth.users |
| asset_type | enum | korean_stock / us_stock / crypto |
| trade_type | enum | buy / sell |
| ticker | text | 종목 코드 또는 코인 티커 |
| name | text | 종목명 |
| trade_date | date | 체결 날짜 |
| quantity | numeric | 수량 |
| price_per_unit | numeric | 체결가 (1주당) |
| currency | text | KRW / USD / 코인티커 |
| fee | numeric | 수수료 |
| fee_currency | text | 수수료 통화 |
| tax | numeric | 세금 |
| settlement_amount | numeric | 정산금액 (실제 입출금액) |
| created_at | timestamptz | |

### 예수금 스냅샷 (`cash_balances`)

| 컬럼 | 타입 | 내용 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | FK → auth.users |
| balance | numeric | 잔고 |
| currency | text | KRW / USD |
| recorded_at | timestamptz | 기록 시점 |

> 예수금은 매매 이벤트의 정산금액 누계로 자동 계산하거나, 사용자가 직접 입력할 수 있다.

## 기술 결정

### 차트 라이브러리

**`react-native-gifted-charts`**
- Expo Go 호환 (react-native-svg 기반, native module 불필요)
- 꺾은선 그래프 + 파이 차트 모두 지원
- 현재도 활발히 유지보수 중

### 외부 API

| 데이터 | API | 무료 한도 | 갱신 주기 | 비고 |
|--------|-----|----------|----------|------|
| 한국 주식 현재가 | **Twelve Data** | 800 req/day | 15분 지연 | KIS Developers(한국투자증권)로 교체 예정 — 계좌 생성 후 |
| 미국 주식 현재가 | **Twelve Data** | 동일 | 15분 지연 | 한국+미국 단일 API로 통합 |
| 코인 현재가 | **CoinGecko** (Demo) | 10,000 req/month | 분 단위 | |
| 환율 (USD/KRW) | **ExchangeRate-API** | 1,500 req/month | 1시간 폴링 | |

> API 호출은 모두 Supabase Edge Function에서 처리. 클라이언트에는 결과값만 전달하여 API 키 노출 방지.

### 한국 주식 API 전환 계획 (추후)

Twelve Data 무료 티어는 15분 지연 데이터. 실시간 현재가가 필요해지면 **KIS Developers (한국투자증권 Open API)** 로 교체. 계좌 개설 필요.

## 평단가 계산 규칙

매수 이벤트 누적 기준 가중평균 매수가.

```
평균매수가 = (기존 보유수량 × 기존 평균매수가 + 금번 매수수량 × 금번 체결가)
             ÷ (기존 보유수량 + 금번 매수수량)
```

매도 시 평균매수가는 변경하지 않는다. 전량 매도 후 재매수 시 새로 계산.

## 비기능 요구사항

- 모든 데이터는 본인 소유만 조회/수정 가능 (RLS)
- 현재가 API 응답 실패 시 마지막 캐시 값 사용, 갱신 시각 표시
- 미국 주식 / 코인 금액은 원화 환산 시 최근 환율 적용, 환율 기준 시각 표시
- 그래프 데이터 없을 시 빈 상태 안내 문구 표시
- CSV 업로드 실패 시 실패한 행 번호와 사유 표시
